# RipioTurismo---RN

### Instalacion

Aca les vamos a mostrar como instalar nuestro proyecto

1. Clona el repositorio
   ```sh
   git clone https://github.com/agusfulco1/Practica-RN.git
   ```
2. ubicate en la carpeta `ParteRN` y pon en la terminal
   ```js
   npm i
   ```
