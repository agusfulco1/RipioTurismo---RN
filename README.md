# RipioTurismo---RN

### Instalacion

Aca les vamos a mostrar como instalar nuestro proyecto

1. Clona el repositorio
   ```sh
   git clone https://github.com/agusfulco1/Practica-RN.git
   ```
2. Ubicate en la carpeta `RipioTurismo---RN` y pon en la terminal
   ```js
   npm i
   ```
3. En la terminal
   ```sh
   expo start
   ```
